# Yinao Personal Homepage

This is a GitHub Pages-ready personal homepage for **yinao**.

## Files

- `index.html`: Main page
- `styles.css`: Styles
- `script.js`: Interactions (theme switch, mobile nav, form demo)
- `_config.yml`: GitHub Pages site config

## Publish to GitHub Pages

1. Create a repository named `yinao.github.io`.
2. Upload all files in this folder to the repository root.
3. In repository settings, open **Pages**:
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
4. Wait a few minutes, then visit:
   - `https://yinao.github.io`

## Customize

- Edit content in `index.html`.
- Replace project cards and contact details.
- If you do not want Jekyll theme settings, remove `_config.yml`.
